# PDFPals - Ultimate Local User Guide

This guide explains how to use the **PDFPals** suite effectively. All processing is 100% local, private, and secure.

## 🚀 Getting Started

1.  **Start the Server**:
    ```bash
    node server.js
    ```
2.  **Access the Dashboard**:
    Open `http://localhost:3000` in your browser.
3.  **Choose Your Theme**: Use the toggle in the top-right to switch between Light, Dark, Terminal, and Ubuntu themes.

---

## 🛠️ Tool Master Class

### ✍️ Sign PDF (Digital Signatures)
- **Place Signature**: Use the 'Sign' button in the sidebar. You can **Draw**, **Type**, or **Upload** your signature.
- **Fields**: Drag and drop fields like **Name**, **Date**, and **Initials** directly onto the document.
- **Adjust**: Click any placed element to move it. Use the handle to resize.
- **Privacy**: Your signatures are never stored on any server; they stay in your browser.

### 📸 CamScanner (AI Digitization)
- **AI Engine**: Wait for the "AI Engine Ready" badge.
- **Detection**: The engine detects document boundaries in real-time.
- **Magic Filter**: Use the **B&W Document (Magic)** filter to turn photos into crisp, high-contrast scans.
- **Manual Crop**: Fine-tune the corners after capture for perfect perspective.

### ⬛ Redact PDF (Security)
- **Selection**: Drag to create blackout zones.
- **Permanence**: Unlike simple "hidden" text, this tool permanently erases the data from the PDF structure.
- **Clear All**: Use the clear button to reset your selections before saving.

### 📥 Asset Ripper (Extract Images)
- **Deep Scan**: Automatically find and extract every embedded image asset in a PDF.
- **Export**: Save individual images or download the entire collection as a `.zip` file.

### 🔗 Organize, Merge & Split
- **Organize**: Drag and drop page thumbnails to reorder or delete pages visually.
- **Merge**: Combine multiple PDFs into a single professional document.
- **Split**: Break large files into individual pages or specific ranges.
